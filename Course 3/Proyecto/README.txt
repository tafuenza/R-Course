The script works as follows:

1- Open the files that are requested, then join them according to their dimensions and have complete information
2- Add both the activity and the subject to each of the observations
3- Replace the number of observations with the real name of each of these
4- Filter the observations by which they have meaning () and std (), then edit the names citing the "," "-" and capital letters
5- The data is grouped by activity and subject to finally create the final file
